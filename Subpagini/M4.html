Origine

Ungaria

Istoric rasa

Este un caine mai putin raspandit ce a aparut la sfarsitul secolului al XIX-lea fiind o rasa relativ tanara. Este recunoscut ca rasa din 1930 si se intalneste destul de rar.

Descriere Fizica

Este un caine de dimensiuni medii cu un corp usor alungit. Capul este lung cu un craniu bombat si cu un bot conic. Ochii sunt ovali, usor oblici si de culoare inchisa. Urechile sunt medii, triunghiulare, ascutite si ridicate. Linia corpului este inclinata spre coada, iar aceasta este scurta si o tine in jos sau poate fi taiata. Parul are 5-7 cm este cret sau ondulat si stralucitor. Pe cap si membre parul este mai scurt si neted. Blana are culoare neagra sau alba sau o combinatie a lor.

Personalitate

Este un caine inteligent, foarte activ, ascultator, invata cu placere, curajos, atent si sobru. Este putin rezistent la efort fizic indelungat. Iubeste familia la care traieste, dar mai ales o anume persoana. Se intelege bine cu alti caini si alte animale. Se intelege si cu copiii dar sunt prevazatori si suspiciosi cu strainii..

Ingrijire

Necesita periaj redus, dar regulat pentru a mentine aspectul frumos al blanii. Periajul va fi mai intens in perioada de naparlire.

Conditii de viata

Nu este un caine pretentios si destul de rezistent la intemperii, trebuie sa aiba posibilitati de miscare si activitati organizate. Are nevoie de socializare, dresaj si exercitii.

Dresaj

Nu este o rasa greu de dresat, ii plac exercitiile sportive cum sunt cele de agilitate si flyball.

Utilitate

Este folosit la paza turmelor de animale si paza locuintelor. Fiind curajos este folosit si la vanatoarea de porci mistreti. Este un bun si placut animal de companie.
