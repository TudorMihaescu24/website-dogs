Origine

Franta

Istoric rasa

Se presupune ca este rezultatul incrucisarilor intre ogarul arab Sloughi si cainii gonitori francezi, vechi prepelicari proveniti din rasa Barbet. A aparut deseori in picturile si tapiseriile franceze. A fost raspandit in toata Franta iar mai tarziu s-a stabilit in Bretania unde a primit si infuzie de sange englezesc, irlandez si scotian de la rasele respective de setteri. Este una din cele mai populare rase de vanatoare.

Descriere Fizica

Este un caine de talie medie, puternic, viguros. Capul este rotund, craniul de lungime medie, botul destul de ascutit, cu nas mare, maro sau roz, dupa culoarea blanii. Ochii sunt de marime medie si de culoare caprui inchis. Urechile sunt scurte, cu varful rotunjit si lasate pe langa cap. Coada este scurta din nastere iar daca este mai lunga se taie la 10 cm, este purtata drept sau atarnata si are in varf un smoc de par. Blana este formata din par fin, des, matasos, usor ondulat, cu franjuri pe gat, burta si partea posterioara a picioarelor. Are culoarea alb si portocaliu, alb cu maro-roscat (cea mai frecventa) alb si negru, tricolor sau rosu, alb si negru.

Personalitate

Este un caine inteligent, exuberant, activ, vioi, simtitor, prietenos, tenace. Este ascultator si devotat stapanului, bland si iubitor cu copiii. Latra putin si doar atunci cand simte pericolul. Se intelege bine cu alti caini si alte animale din gospodarie.

Ingrijire

Un periaj ocazional este suficient pentru intretinerea blanii, pentru a nu se incalci.

Conditii de viata

Este un caine cu un ridicat nivel de activitate si se simte bine langa un stapan activ, care isi petrece mult timp in natura. Ii place sa aiba spatiu ingradit de miscare, sa participe la sporturi canine, cum ar fi concursuri de agilitate si flyball, sa inoate, sa faca aport. Are nevoie de dresaj.

Dresaj

Este un caine care se dreseaza usor pentru ca este ascultator, inteligent, invata repede. Cu un ton calm, cu o voce blanda, o atitudine consecventa se pot obtine rezultate bune.

Utilitate

Este un excelent pontator si aportor. Vaneaza in padure si desis. Este un placut companion.
