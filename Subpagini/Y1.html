Origine

Anglia

Istoric rasa

Este o rasa veche englezeasca care a aparut prin incrucisarea Manchester Terrierlui cu Skye Terrier si cu Dandie Dinmont Terrier. A fost crescut de minerii din ducatul York. La acea rasa au fost aduse imbunatatiri, obtinandu-se exemplarele actuale, foarte apreciate si iubite atat in Anglia cat si in alte tari. Initial, a fost folosit la vanatoarea de soareci si sobolani in minele si morile din Yorkshire. Apoi a devenit animal de companie la moda in epoca Victoriana. Denumirea actuala a fost data in 1870.

Descriere Fizica

Este un caine de talie mica, unele exemplare chiar foarte mici, bine proportionat. Are capul mic si plat, botul de lungime egala cu craniul, conic, maxilarul puternic si acoperit cu par foarte lung, nasul negru si mic. Ochii sunt de marime medie si de culoare inchisa. Urechile sunt mici, in forma de V, ridicate si acoperite de par scurt. Coada este taiata la jumatate si purtata ridicat, peste nivelul spatelui si acoperita de un par bogat. Blana deosebita, este formata dintr-un par foarte lung, drept, matasos, lucios si foarte fin. Puii se nasc negri cu cafeniu si abia la 2 ani vor avea culoarea caracteristica de maro-auriu cu gri-metalic. Parul de pe cap si mai des de pe frunte este de bogat si lung incat trebuie sa fie strans si legat deasupra capului pentru a degaja ochii. Blana se poarta cu o carare pe mijlocul spatelui.

Personalitate

Este un caine curajos si hotarat, inteligent, vioi, vesel, alert si vigilent. Este un caine sensibil, trebuie tratat cu blandete, afectiune. Este ascultator si devotat stapanului. Se intelege bine cu copiii atat timp cat nu il vor necaji prea mult . Se intelege bine si cu alte animale din gospodarie. Oaspetii vor fi intotdeauna anuntati si latra cand este pericol. Unele exemplare pot avea o atitudine imprudenta fata de alti caini.

Conditii de viata

Este un excelent caine de apartament. Trebuie tratat cu multa dragoste, afectiune fiind un caine sensibil.Are un nivel de actvitate moderat si scurte plimbari cat de des se poate ii vor face placere. Se adapteaza si cu un stapan mai putin activ.

Dresaj

Este o rasa care se dreseaza usor.Dresajul trebuie sa fie consecvent, variat, facut cu blandete si multa dragoste incat sa-i faca placere cainelui si sa faca tot ce-i cere stapanul.

Utilitate

Este un bun paznic si un placut si afectuos companion.
