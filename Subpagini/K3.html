Origine

Ungaria

Istoric rasa

Rasa provine din cainii tibetani adusi in Ungaria de ciobanii din triburile nomade in sec al IX-lea. Este raspandit in mai multe tari din Europa si in SUA unde este foarte apreciat.

Descriere Fizica

Este un caine de talie mare cu un corp patrat, puternic, musculos. Capul este mare, proportional cu corpul, cu ochi migdalati de culoare inchisa, acoperiti de par. Urechile sunt mari, bogate in par si purtate atarnat. Coada este lunga, usor intoarsa la varf, purtata atarnat, iar in stare de agitatie se ridica cel mult la nivelul spatelui. Blana este deosebita, foarte deasa si lunga, dandu-i asepectul de masivitate. Este intotdeauna alba, dubla, cu un prim strat pufos, foarte des, lanos si impermeabil. Al 2-lea strat este lung, paslos si incretit si poate fi sub forma de snur.

Personalitate

Este un caine intelignent, vigilent, foarte curajos, atacand ursii si lupii. Tinuta masiva trezeste teama. Este calm, echilibrat si nu latra prea mult, atacand pe neasteptate. Este deosebit de credincios stapanului, dar ataca pe straini. Nu se intelege cu alte animale de pe teritoriul sau. Se intelege relativ bine cu copiii din familie.

Ingrijire

Blana acestei rase nu se perie, dar trebuie smulse cu mana smocurile care se formeaza din puful ce cade daca vrem sa se formeze o blana sub forma de franjuri. Baia se recomanda doar vara deoarece ii trebuiesc mai multe zile pentru a se usca.

Conditii de viata

Putin pretentios, este rezistent la intemperii. Are nevoie de un stapan hotarat si cu experienta in cresterea cainilor. Necesita spatii pentru miscare, socializare, dresaj.

Dresaj

Dresajul trebuie sa fie ferm si consecvent. Intelege greu, dar odata inteles un ordin nu-l va incalca. Nu este potrivit pentru activitatile de dresaj sau alte sporturi.

Utilitate

Este un excelent caine da paza al turmelor de vite si al locuintelor.
