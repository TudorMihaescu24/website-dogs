Origine

Norvegia

Istoric rasa

Este o rasa foarte veche si dupa unii specialisti descinde din rasa Iceland Dog. A fost folosit ca animal bun la toate (paza, pastorit sau vanatoare). Este cunoscut si in alte tari ca Anglia, Australia unde este foarte apreciat, fiind pe plan international din 1970.

Descriere Fizica

Este un caine de talie medie, cu forma corpului patrata, bine construit. Are un cap lat, triunghiular, botul usor conic, cu nasul si ochii negri si seamana cu o vulpe. Urechile sunt medii spre mari, triunghiulare cu varfurile rotunjite situate in partea din spate a craniului si foarte mobile. Coada este scurta, groasa si tinuta pe spate. Blana este dubla, cu un prim strat pufos, moale, scurt si cu un strat exterior care are un par aspru, des, lung si neted. Culorile sunt galben-pai, crem sau (mai rar) negru cu putin alb.

Personalitate

Este un caine inteligent, curajos, energic, zgomotos, cu simturi (auz, vaz, miros) foarte dezvoltate, este vigilent si atent. Devotat si afectuos cu familia, accepta copiii. Se intelege bine cu alti caini si alte animale din gospodarie. Strainii sunt anuntati, dar numai atat.

Conditii de viata

Este un caine rezistent la intemperii care se simte bine in aer liber si in zone cu un climat mai rece langa o familie activa. Are nevoie de activitate si miscare. Are nevoie de socializare si dresaj.

Dresaj

Este un caine inteligent si dornic sa invete si de aceea se dreseaza destul de usor. Cu un dresaj consecvent, ferm, variat si bland se pot obtine rezultate bune. Ii place miscarea, sporturile canine, se bucura sa faca aport, sa alerge pe langa bicicleta.

Utilitate

Este folosit la paza turmelor de vite, la vanatoare sau ca animal de companie.
