Origine

Japonia

Istoric rasa

Este cel mai mic caine de Japonia si cea mai veche dintre rasele japoneze. Are infatisarea neschimbata de 1800 ani. Este numit si caine de desis pentru ca era folosit la vanatoarea pasarilor. Astazi este folosit mai mult ca animal de companie.

Descriere Fizica

Este un caine de talie mica spre mediu (Shiba Inu in japoneza inseamna caine mic), cu aspect asemanator unei vulpi, cu un corp musculos, robust. Capul este proportional cu corpul. Fruntea este lata cu stop moderat, bot conic, maxilare puternice si nas negru. Ochii sunt mici, ovali, de culoare inchisa. Urechile sunt mici, distantate, triunghiulare, ridicate. Coada este de lungime medie, stufoasa si tinuta pe spate. Blana este dubla, cu un prim strat des, scurt, pufos si un strat exterior cu un par scurt, aspru drept. Cele mai des intalnite culori sunt : rosu, rosu cu zone albe, negru, alb, negru cu cafeniu, sare si piper.

Personalitate

Este un caine inteligent, curios, alert, vioi, plin de viata, vesel, vigilent. Uneori independent, ii mai place sa plece singur. Afectuos si devotat stapanului, nu are probleme cu copiii si este un pic rezervat cu strainii. Se intelege cu alti caini si alte animale.

Conditii de viata

Este un caine adaptabil atat in curte cat si in apartament. Are un nivel mediu de activitate, se va adapta si dupa un stapan mai putin activ. Ii plac plimbarile lungi si face fata la ele. Are nevoie de socializare, dresaj.

Dresaj

Se dreseaza usor pentru ca este un caine inteligent si invata repede. Se obtin rezultate mai bune daca se abordeaza o metoda mai blanda, cu nuante jucause. Trebuie dresat de mic pentru a transforma caracterul exuberant specific rasei intr-un caracter temperat, docil. Se descurca bine la competitiile de agilitate.

Utilitate

Este utilizat mai rar la vanatoare de animale mici si mai mult ca animal de companie.
