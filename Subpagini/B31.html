Origine

Marea Britanie

Istoric rasa

Este o rasa foarte cunoscuta si iubita in Anglia. Originile indepartate nu sunt prea cunoscute. Una din ipotezele emise presupune ca bulldog-ul descinde din Molossul din Epir, stramosul comun al dog-ilor. Prima mentiune despre Bulldog este din 1209 in descrierea unei lupte intre cainii macelarilor si un taur. Lupte sangeroase intre Bulldogi si alte animale (tauri, ursi, urangutani) au fost permise in Anglia pana in 1835. Chiar daca acesti caini au fost apoi mai putin cautati, totusi, fanii acestei rase au perpetuat-o panstrandu-i puterea si curajul, dar eliminandu-i apucaturile agresive nedorite si punand accent pe temperamentul bland si aspectul distinctiv.

Descriere Fizica

Este un caine de talie medie, masiv, cu un corp greu, indesat, sustinut de picioare scurte, puternice. Are un cap masiv, lat, patratos, cu un craniu scurt, frunte plata si ridata, sprincene proeminente. Botul este scurt, lat si comprimat, cu maxilarul inferior mult iesit in afara fata de cel superior, un nas carn si buzele groase, mari si atarnate. Ochii sunt adanciti in orbite, departati, rotunzi si de marime medie. Urechile sunt departate, mici, atarnate si cu o portiune din partea interna a pavilionului vizibila. Gatul este scurt, gros, iar pieptul este lat si adanc. Parul este scurt si fin, neted si lucios, bine lipit de corp si de culoare bej, maron-roscat, dungat (cu sau fara fata neagra), alb si baltat.

Personalitate

Bulldogul de astazi este un caine vioi, spontan, curajos, echilibrat, inteligent, bland si sensibil. Credincios si ascultator cu stapanul, este tolerant si grijuliu cu copiii. De obicei se intelege bine cu alte animale, uneori poate fi certaret cu alti caini, dar de regula se intelege cu ei.

Conditii de viata

Este o rasa adaptata foarte bine la viata de apartament. Cu un nivel scazut de activitate, este multumit cu plimbari scurte zilnice, si in rest ii place sa stea langa familie. Este recomandat persoanelor mai putin active. Necesita dresaj si socializare.

Dresaj

Este un caine usor de dresat. Se recomanda ca dresajul sa fie facut pe un ton bland, prietenos, dar ferm fara a fi sever. Trebuie sa fie consecvent si echilibrat.

Utilitate

Este un bun caine de paza si de companie.
