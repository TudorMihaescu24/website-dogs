Origine

Brazilia

Istoric rasa

Aceasta rasa este considerata a fi o incrucisare intre cainii Bloodhound, Mastiff si Bulldogul englez si a fost creata de cuceritorii spanioli pentru a prinde sclavii evadati.

Descriere Fizica

Este un caine de talie mare, mai mult lung decat inalt, viguros. Capul este masiv, cu botul mare, puternic, cu nas negru si lat, buze groase si lasate. Ochii sunt migdalati, potriviti ca marime si de culoare galben deschis pana la caprui inchis. Urechile sunt in forma de V, mari, lasate natural. Gatul este puternic, iar coada este lata la baza si acutita la varf. Blana are un par scurt, lipit de corp, drept si poate fi orice culoare in afara de alb, poate fi si tigrat cu pete albe pe piept, gat, coada si extremitatile membrelor.

Personalitate

Este un caine curajos, activ, inteligent, cu un miros fin, cu puternice instincte teritoriale si destul de agresiv. Este devotat si ascutator cu stapanul, fidel, tolerant cu familia, inclusiv copiii. Este suspicios si rezervat cu musafirii si neincrezator, chiar agresiv cu strainii. Accepta numai animalele cu care a fost socializat de mic, cele nou aparute nu vor fi accepate. Este considerat un caine periculos.

Ingrijire

Acest caine nu necesita prea multa ingrijire. Periajul blanii ocazional si mai ales in perioada de naparlire este suficient.

Conditii de viata

Este recomandat sa fie crescut numai de o persoana cu multa experienta in cresterea animalelor, intr-o curte ingradita pentru a putea face miscare. Are nevoie obligatoriu de dresaj si socializare.

Dresaj

Trebuie dresat de o persoana cu multa experienta care sa stie cum sa il faca pe caine sa inteleaga cine este stapanul. Fiind un caine cu un caracter mai dificil, dresajul trebuie sa fie ferm, fara a fi sever, facut cu calm, rabdare caci este sensibil la tonul vocii. Este un caine care reactioneaza rapid si cu instincte de agresivitate innascute. Cu toate acestea daca dresajul este facut corespunzator si are un stapan potrivit, el poate fi foarte devotat si ascultator.

Utilitate

Este folosit ca paznic pentru vite, la vanatoarea de animale mari, pentru luat urma sau ca un excelent paznic al locuintei.
