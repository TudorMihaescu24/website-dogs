Origine

Germania

Istoric rasa

Este o rasa relativ noua alcatuita prin incrucisarea dintre Chow-Chow, Wolfspitz si Samoyed. Prin selectii repetate si severe s-au obtinut exemplarele de astazi care sunt apreciate atat in Europa cat si in SUA.

Descriere Fizica

Este un caine de talie medie, mai mult lung decat inalt, musculos. Capul este puternic, asemanator cu al lupului, de forma conica, cu craniul plat, cu o cuta frontala, cu un bot conic si maxilarele puternice. Ochii sunt de marime medie, oblici si de culoare inchisa. Urechile sunt de marime medie, ascutite, triunghiulare, rotunde la varfuri si ridicate. Coada este groasa, stufoasa si purtata pe spate. Blana este dubla, cu un prim strat des, scurt, pufos si un strat exterior cu par drept, de lungime medie si aspect ciufulit. Parul este scurt pe bot, fata si partea anterioara a picioarelor si lung in rest. Poate fi orice culoare in afara de alb si rosu-inchis.

Personalitate

Este un caine inteligent, calm, nu latra prea mult, echilibrat, alert, destul de prietenos si ascultator, uneori cu o gandire independenta. Este afectuos si devotat familiei, se intelege destul de bine cu copiii, dar precaut si rezervat cu strainii. In general nu are probleme cu alti caini si nici cu alte animale din gospodarie, mai ales daca a fost invatat de mic cu ele.

Ingrijire

Trebuie periat si pieptanat mai des in perioada de naparlire. In rest se pieptana numai ocazional daca este nevoie, stratul inferior ii poate fi afectat daca este prea des pieptanat.

Conditii de viata

Este un caine rezistent la intemperii caruia ii place viata in aer liber alaturi de in stapan iubitor (nu ii place sa stea mult timp singur) dar cu experienta in cresterea cainilor. Se poate adapta si la conditii de bloc numai sa i se asigure zilnic minim o ora de plimbare, sa poata alerga si sa se joace. Are nevoie de socializare si dresaj.

Dresaj

Este un caine care se poate dresa relativ usor pentru ca este inteligent si destul de ascultator. Cu un dresaj ferm (uneori poate fi independent), consecvent, echlibrat, se pot obtine rezultate bune. Trebuie socializat de mic cu alti caini, alte animale si cu copiii.

Utilitate

Este un bun caine de paza al locuintei si un devotat companion.
