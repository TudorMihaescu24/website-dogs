Origine

Islanda

Istoric rasa

Este singura rasa nativa a acestei tari. A fost adusa la sfarsitul secolului al IX-lea de catre vikingi si a fost folosita de catre fermieri pentru paza turmelor.

Descriere Fizica

Este un caine de talie mica spre medie, mai mult lung decat inalt, musculos. Capul este mai lat, format triunghiular, cu un bot conic, scurt si un nas negru. Urechile sunt de dimensiuni medii, triunghiulare, cu varful rotunjit si ridicate. Coada este stufoasa si tinuta pe spate. Blana este dubla, cu un prim strat scurt, des, moale si unul exterior cu un par des, lucios scurt sau lung (sunt doua varietati). Blana este mai lunga la gat, partea posterioara a picioarelor si coada. Poate fi cafeniu, gri, maro sau negru cu pete albe.

Personalitate

Este un caine inteligent, echilibrat, curajos, vigilent, atent, sociabil. Este foarte atasat si devotat fata de familie, sufera cand este lasat singur. Se intelege bine cu copiii si este destul de sociabil chiar cu strainii. Se impaca bine cu alti caini si alte animale din gospodarie.

Ingrijire

Blana acestui caine trebuie periata in perioada de naparlire pentru a indeparta parul mort.

Conditii de viata

Este un caine rezistent la intemperii si se simte bine in aer liber alaturi de un stapan cu experienta in cresterea cainilor. Are nevoie de spatiu ingradit pentru a nu pleca la vanatoare cand simte mirosul vanatului, de botnita cand iese pe strada pentru a nu ataca alti caini (daca nu a fost suficient socializat).

Dresaj

Este un caine care se dreseaza destul de usor pentru ca este inteligent si dornic sa invete. Cu un dresaj consecvent, variat, ferm (pentru a invinge unele tendinte spre incapatanare) si totodata bland se pot obtine rezultate bune. Poate participa si la concursurile de agilitate si flyball.

Utilitate

Este folosit la paza si manatul turmelor si ca animal de companie.
