Origine

Anglia

Istoric rasa

Aceasta rasa a fost creata la mijlocul secolului al XIX-lea si este rezultatul incrucisarilor dintre vechiul terrier englezesc si copoiul Otterhound. A fost dezvoltat in valea Aire din comitatul Yorkshire, de unde ii vine si numele. Este cel mai mare terrier britanic si a fost folosit in diverse scopuri cum ar fi: vanatoare, paza, protectie, ajutor pentru Crucea Rosie in ambele Razboaie Mondiale, luat urma etc. Cateodata acest caine mai este numit regele Terrierilor. datorita aspectului sau care aduce cu al maimutei.

Descriere Fizica

Este un caine de talie mare, robust, musculos, suplu, cu o tinuta eleganta. Capul este lung, plat, pronuntat. Ochii sunt mici, de culoare inchisa si foarte expresivi. Urechile sunt mici, in forma literei V si tinute lateral, cu linia de rasfrangere putin deasupra liniei capului. Coada este sus si purtata aproape vertical, taiata 2/3 din lungime. Blana este dubla cu un strat exterior scurt, tare, sarmos, putin ondulat fara a fi cret. Culoarea este cafenie cu spatele (ca o “sa”) de culoare neagra sau cenusie.

Personalitate

Este un caine inteligent, foarte curajos si vioi, tacut, activ, uneori jucaus si incapatanat, nu latra prea mult. Foarte devotat stapanului, este rabdator cu copiii. Accepta alte animale din gospodarie, dar manifesta tendinte de dominare fata de alti caini daca nu a fost invatat sa-i accepte. Este alert in prezenta strainilor.

Ingrijire

Acest caine nu necesita prea multa ingrijire. Blana va fi trimata de doua ori pe an. Cand este necesar se va taia parul dintre pernutele de la picioare.

Conditii de viata

ste un caine rezistent la intemperii si-i place sa traiasca in aer liber, intr-un spatiu mare, sa poata face multa miscare pentru ca are un nivel ridicat de activitate, langa un stapan activ, tip sportiv. Are nevoie de dresaj, socializare.

Dresaj

Este un caine relativ usor de dresaj, pentru ca este inteligent si receptiv. Dresajul trebuie sa fie consecvent, ferm, fara sa fie dur, variat pentru a nu se plictisi caci devine incapatanat. Ii plac sporturile canine.

Utilitate

Este folosit la vanatoare de animale salbatice (vidre, jderi, vulpi, iepuri, mistreti, ursi). Este un bun caine de paza, iar in Anglia este cel mai folosit caine de catre politie. Este un placut si cautat caine de companie.
