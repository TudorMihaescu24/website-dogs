Origine

Olanda, Germania

Istoric rasa

Aceasta rasa dateaza din secolul al XVI-lea si se considera ca descinde din Wolfspitzul german. Numele vine de la porecla Kees data de Cornelius de Gyslear, conducatorul unei rascoale impotriva Casei Regale de Orania. De Gyslear a folosit acest caine ca simbol al partidului din care facea parte. In continuare a fost folosit pentru paza incarcaturilor pe barje, fiind cunoscut si sub numele de caine olandez de barja. Este rasa nationala a Olandei.

Descriere Fizica

Este un caine de talie medie, robust, proportional. Capul seamana cu al vulpii, are un format triunghiular, cu un stop proeminent, bot de marime medie si nas negru. Ochii sunt migdalati, oblici, caprui-inchis. Urechile sunt mici, situate sus, triunghiulare si ascutite. Coada este stufoasa si tinuta pe spate. Blana este dubla cu un prim strat pufos, des, scurt si un strat exterior cu un par foarte des si lung. Blana este foarte deasa in zona gatului formand o coama. Blana mai este foarte deasa si pe piept, picioarele posterioare, umeri si zona fundului. Are culoarea gri-argintiu, crem si negru complet sau alb.

Personalitate

Este un caine inteligent, dornic sa invete, galagios, prietenos, vigilent, atent. Este loial stapanului si familiei acestuia, accepta copiii si alti caini. Trebuie socializat de mic cu pisicile si alte animale mici din casa. Strainii sunt latrati.

Conditii de viata

Este un caine rezistent la intemperii caruia ii place sa traiasca in aer liber, intr-un climat mai rece, dar se poate adapta si la viata intr-un apartament daca i se asigura nivelul moderat de activitate de care are nevoie.

Dresaj

Este un caine care se dreseaza usor pentru ca este inteligent si invata ce i se cere. Cu un dresaj bland, consecvent, variat se pot obtine rezultate foarte bune putand participa cu succes la cursele de agilitate sau alte sporturi canine.

Utilitate

A fost folosit pentru distrugerea animalelor daunatoare si paza. Astazi este si un placut animal de companie.
