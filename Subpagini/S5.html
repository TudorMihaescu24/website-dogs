Origine

Germania

Istoric rasa

Cainele original al acestei rase este schnauzerul standard. Este rezultatul incrucisarilor dintre Schnauzerul mijlociu, Dogul German si Bouvierul de Flandra.

Descriere Fizica

Este un caine de talie mare, puternic, bine proportionat, cu o infatisare impunatoare care inspira respect. Seamana foarte bine cu schnauzerul standard doar ca este mai mare. Are un cap dreptunghiular, puternic cu botul egal cu craniul. Nasul este rotund si negru. Ochii sunt de marime medie, ovali si de culoare inchisa. Urechile de lungime medie, in forma de V sunt atarnate din nastere si pot fi taiate pentru a sta ridicate. Are sprancene stufoase, mustati si barba caracteristice acestei rase. Coada este taiata scurt si purtata ridicata cand este in alerta si orizontal cand este in repaos. Blana este dubla, cu un strat exterior cu par aspru, des si de culoare neagra, sare si piper, gri-fier pana la gri argintiu.

Personalitate

Este un caine foarte curajos, inteligent, calm, muncitor, destul de agresiv cu o rezistenta fizica foarte buna. Este protector, devotat, bland chiar jucaus cu familia, dar poate deveni putin agresiv daca este provocat. Se ataseaza de stapan si nu se indeparteaza mult de acesta. In general sunt toleranti cu copiii. Socializat corect de mic ajunge sa accepte in preajma alti caini sau alte animale din gospodarie, in caz contrar pot aparea probleme.

Conditii de viata

Este un caine rezistent la intemperii caruia ii prieste viata in aer liber alaturi de un stapan activ si cu experienta in cresterea cainilor. Necesia dresaj, socializare, exercitii.

Dresaj

Necesita un dresaj corect, consecvent si variat facut de o persoana experimentata.Trebuie socializat de mic cu copiii, alte animale si alti caini. Iubeste miscarea si simte nevoia de activitate.

Utilitate

A fost creat pentru a fi folosit la manarea vitelor si la paza fermelor. A fost folosit in primul razboi mondial de catre armata. Este un excelent paznic al fermelor, al grajdurilor cu cai, deoarece se intelege foarte bine cu aceste animale, un bun aparator al locuintelor, un devotat companion. Este folosit de armata si politie deoarece executa cu placere orice sarcina.
